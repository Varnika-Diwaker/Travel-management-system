# Travel-management-system
The Travel Management System is a Java-based application designed to automate and simplify travel-related operations such as booking management, customer details, and tour packages. The system uses MySQL as the backend database to store and manage data securely and efficiently.This project demonstrates the practical use of Java, JDBC, and MySQL .

#Key Features

User login and authentication

Manage customer details

 view travel packages

Booking management system

Secure database connectivity using JDBC

Simple and user-friendly interface

#Technologies Used

Programming Language: Java

Database: MySQL

Database Connectivity: JDBC

IDE: NetBeans 

Platform: Desktop Application
#Objective

The main objective of this project is to provide a centralized system for managing travel information efficiently while reducing manual work. It also helps learners understand database connectivity, CRUD operations, and structured application design in Java.
